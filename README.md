# Smart_Store_Inventory_Optimization_For_Retail_Stores
This project is developed as part of the Infosys Springboard Internship.
The goal is to help retail stores optimize inventory.
A data-driven Smart Store Inventory Optimization System using Python, Pandas, NumPy, Matplotlib, Plotly, and Streamlit. Helps retail stores optimize stock levels, forecast demand, and visualize sales trends
